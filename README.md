# 工大圈子

哈工大软件工程项目：Android blog app

## 架构

开发环境： Android studio

语言：Java

数据库：阿里云OSS关系数据库

## 功能

* 博客功能：浏览博客，在线刷新，发布博客
* 好友聊天：类似微信聊天界面在线发送接收信息
* 好友功能：好友列表
* 个人信息：头像设置，信息修改
* 登陆：密码修改等
